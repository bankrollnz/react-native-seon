# react-native-seon

React Native Bridge for the Seon Fruad Fringerprint SDK

## Installation

```sh
npm install react-native-seon
```

## Usage

```js
import Seon from "react-native-seon";

// ...

const result = await Seon.multiply(3, 7);
```
